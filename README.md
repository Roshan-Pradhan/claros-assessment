# Claros Analytics Assessment

This project is a React-based frontend application built for the Claros Analytics assessment.

## Tech Stack

- **React**
- **TypeScript**
- **Tailwind CSS**
- **Redux Toolkit**
- **React Query**
- **React Table**
- **React Paginate**
- **React Testing Library + Vitest**

## Installation

Make sure you have **Node.js** and **npm** installed.

```bash
npm install
```

Run Test

```bash
npm test
```
